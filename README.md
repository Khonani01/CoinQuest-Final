# CoinQuest - Personal Budget Tracker
**Module:** OPSC6311 - App Prototype Development

## Overview
CoinQuest is a gamified personal budgeting application built using Kotlin and Android Studio. It utilizes an MVC architecture and a Room offline database for data persistence.

## Features Included:
* Local SQLite data persistence (Room DB)
* User Authentication
* Category and Expense logging (with start/end times and receipt photos)
* Gamification (XP and Badges)
* Visual analytics (Bar and Donut charts)

## CI/CD
This repository uses GitHub Actions for automated building and testing.
